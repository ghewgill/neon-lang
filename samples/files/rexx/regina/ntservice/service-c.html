main

void _CRTAPI1 main(int argc,
char **argv)

PURPOSE: entrypoint for service

PARAMETERS

argcnumber of command line arguments
argvarray of command line arguments

RETURN VALUE

none

COMMENTS

main() either performs the command line task, or call StartServiceCtrlDispatcher to register the main service thread.  When the this call returns, the service has stopped, so exit.
