The Neon documentation is split into several parts.

  • Manifesto. Why does Neon exist? What problems does it try to solve?
  • Tutorial. This is an introduction to the Neon language.
  • Standard Library. This describes every module in the Neon Standard Library.
  • Overview for Experts. This is a very quick overview of the language for experienced programmers.
  • Language Reference. This is a detailed reference describing every language feature.
  • Grammar Diagrams. Detail of the language grammar using “railroad diagrams”.
  • Samples. This describes a number of sample programs written in Neon.
